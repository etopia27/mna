Project specificilation/requirements -

1. JDK 1.8
2. Maven 3 or higher version

Technology stack used -

Spring Security
Spring Boot
Spring Data JPA
JSP

Database used as default HSQL- later will use MYSQL.


Steps to run application- 

1. build and start application using
	mvn clean spring-boot:run
	
2. visit to http://localhost:8080
3. login details 
	userName - admin
	password - 123456
4. on successful login user will land file upload tab
	i. For Poc you can upload max two files and will get message below button
    ii.You can switch to Project tab, which will show listing of project having some dummy data for POC purpose only
